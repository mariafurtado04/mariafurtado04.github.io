# Memory-Loss
Memory Loss is an interactive multimedia experience created in an academic context as part of a project that challenged students to develop an interactive object exploring the theme of memory. The project explores the fragility of memory and the emotional impact of losing it, inviting the user to actively participate in a narrative that represents a mind affected by memory deterioration.

The concept was inspired by scientific studies suggesting that both physical and mental activity can help slow memory decline and support cognitive function. Based on this idea, the project transforms the user’s movement into a metaphor: the preservation of memory depends on active participation.

The experience features fragments of an image floating across the screen, symbolizing scattered memories. These fragments only begin to reconnect when the participant moves. If the user remains still, the fragments gradually disappear, representing the progressive loss of memory and the confusion associated with it. Through a minimalist visual language and subtle sound elements, the project encourages reflection on the fragility of memory and the importance of remaining mentally and physically active.
